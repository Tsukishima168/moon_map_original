## 三、完整的推廣優先順序

### 第 0 步：完成訂購系統（必須優先）

```
時間點：現在到下個月
狀態：80% 完成

理由：
✓ 訂購系統是「變現」的最後一哩路
✓ 其他所有推廣都是為了把人導到訂購系統
✓ 沒有訂購系統，再多流量也沒用

優先完成清單：
□ 修復 bug 和測試用例
□ 優惠券系統運作順暢
□ LINE 整合（能直接從 LINE 預訂）
□ 付款整合（信用卡、LINE Pay、Pi）
□ 會員帳號整合（與集點卡互通）
□ 舊客測試反饋和改進
□ 設定好「第一批優惠券」（吸引初期用戶）

預計完成時間：1-2 周
上線時間：下周

❌ 不要等訂購系統完全完美才推廣
✅ 80% 完成就可以小範圍推廣給舊客
✅ 邊推廣邊優化
```

## 九、常見問題解答

### Q1：訂購系統什麼時候上線？

```
建議：2 周內
理由：
□ 已經 80% 完成
□ 舊客內測 1-2 周
□ 修復反饋 3-5 天
□ 正式推廣

不要等完美，邊推廣邊優化
```

### Q2：應該同時推廣還是逐一推廣？

```
答案：有優先級的同時推廣

Week 1-2：訂購系統 + 舊客內測
Week 3-4：Google Maps + IG + Linktree 同步推廣
Week 5+：所有管道一起運營

原因：
□ 越早推廣，越快見效
□ 不同平台有不同粉絲
□ 同時推廣能製造「到處都看得到」的效果
□ 但優先級明確：Google Maps > IG > 店內 > 其他
```

### Q3：哪個平台 ROI 最高？

```
ROI 排行：

🥇 Google Maps（菜單連結）
   時間投入：30 分鐘
   月預期訂單：25-50
   ROI：最高

🥈 店內推廣（QR code）
   時間投入：2 小時（一次性）
   月預期訂單：50-100
   ROI：非常高

🥉 IG Stories
   時間投入：每周 20 分鐘
   月預期訂單：20-40
   ROI：高

其他都是中等或輔助性質
```

### Q4：應該投放 Google 廣告嗎？

```
答案：短期不用

理由：
□ 有機管道還有很大空間
□ Google Maps + IG 的轉化率已經不錯
□ 先把有機優化到 80%，再考慮廣告

什麼時候投廣告：
□ 3 個月後，有機訂單達到 150+ 筆/月
□ 預算充足且有明確 ROI 計算
□ 作為「衝刺季度銷售」的工具

投廣告的優先順序：
1. Google Local Services Ads（評論充足後）
2. IG Ads（粉絲基數足夠後）
3. Facebook Ads（目標客群明確後）
```

### Q5：舊客內測要多久？

```
建議：2-3 周

第 1 周：收集 20-30 人的使用反饋
第 2 周：修復 bug，再測試 20-30 人
第 3 周：確認沒問題，開始全員推廣

預期：
□ 收集 10-20 筆成功訂單
□ 找出 3-5 個最常見的問題並修復
□ 蒐集 3-5 個「用戶見證」（用於推廣）
```
