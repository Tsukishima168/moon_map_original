# Project Overview

Project: Kiwimu ecosystem (local canonical repo: map-kiwimu-com; GitHub repo: moon_map_original)

Sites:
- MBTI Lab (kiwimu.com)
- Shop (shop.kiwimu.com)
- Moon Map (map.kiwimu.com)
- Passport (passport.kiwimu.com)
- Gacha (gacha.kiwimu.com)

Stack:
- React / Next.js
- Supabase (DB + Auth + RLS)
- Vercel
- n8n
- GA4 (ID: G-DM6F27KL8B)

Architecture principle:
- SOC (Separation of Concerns)
- 3-Tier Layer: Route (Controller) -> Service -> Repository
- Event Driven via EventBus

Phases:
- Phase 1 → Traffic validation
- Phase 2 → Commerce scaling
- Phase 3 → Ecosystem integration
